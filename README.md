# QQ拟人化插件 (astrbot_plugin_qq_humanize)

> QQ 个人号拟人化与群聊权限管理插件，降低腾讯风控检测封号风险

## 功能特性

### 1. 拟人化延迟
- 消息发送前添加随机延迟，模拟真人打字行为
- 延迟严格控制在 **5 秒以内**
- 根据文本长度动态计算打字速度（每字 0.03-0.1 秒）

### 2. 群聊权限管理
三种模式可设置：
| 模式 | 说明 |
|------|------|
| 所有人 | 任何人都能和机器人聊天 |
| 管理员 | 群管理员 + 白名单用户可聊 |
| 白名单 | 仅白名单用户可聊 |

### 3. 聊天记录存储
- SQLite 数据库自动存储群聊消息
- 保留 24 小时，过期自动清理
- 第二天自动删除前一天记录

### 4. AI 聊天总结
- 调用 AstrBot 内置 LLM 总结群聊内容
- 总结回复也遵守拟人化延迟

### 5. 无前缀指令
- 所有指令无需 `/` 前缀，直接发送即可
- 支持中文指令和英文指令
- 管理员可用大白话管理

### 6. 私聊黑名单
- 管理员可禁止指定QQ私聊机器人
- 被禁止的用户私聊将不会得到回复
- 支持大白话：`@机器人 禁止123456私聊`

### 7. 点赞功能
- 给指定用户QQ资料卡点赞
- 需NapCat/Lagrange协议端支持
- 每天每人10次点赞限制

---

## 安装方法

1. 下载 `astrbot_plugin_qq_humanize.zip`
2. AstrBot WebUI → 插件管理 → 上传插件 → 选择 zip
3. 安装完成，发送「帮助」查看说明

## 使用说明

> 所有指令无需 `/` 前缀，直接在群里发送即可

### 帮助
```
帮助
```

### 群聊权限管理（管理员）

**指令方式：**
```
设置模式 所有人      设置为所有人可聊
设置模式 管理员      设置为仅管理员可聊
设置模式 白名单      设置为白名单模式
加白 123456          添加QQ到白名单
移除 123456          从白名单移除
群状态                查看当前群设置
```

**大白话方式（@机器人）：**
```
@机器人 所有人都能聊
@机器人 只有管理员能聊
@机器人 把123456加入白名单
@机器人 禁止789聊天
```

### AI 总结（所有人）
```
总结            总结最近24小时聊天
总结 6          总结最近6小时
```

也可以 @机器人 说：
```
@机器人 总结一下今天的聊天
@机器人 今天聊了什么
```

### 拟人化设置（管理员）
```
拟人化配置              查看当前配置
设置延迟 0.5 3          设置延迟范围 0.5-3秒
```

> 延迟最大值不超过 4.8 秒，加上抖动后总延迟严格 < 5 秒

### 私聊黑名单（管理员）
```
禁私聊 123456           禁止该QQ私聊机器人
解禁私聊 123456         解除私聊禁止
私聊黑名单              查看被禁止的QQ列表
```

大白话方式（@机器人）：
```
@机器人 禁止123456私聊
@机器人 不让789私聊
```

### 点赞（管理员）
```
点赞 123456             给指定QQ资料卡点赞
点赞                    给自己点赞（群聊中）
```

大白话方式（@机器人）：
```
@机器人 给123456点赞
@机器人 赞一个123456
```

> 需NapCat/Lagrange等协议端支持，每天10次

## 配置文件

配置文件位于插件目录 `config.json`：

```json
{
  "group_settings": {},
  "humanize_settings": {
    "min_delay": 0.5,
    "max_delay": 4.5,
    "random_typing_delay": true
  }
}
```

## 数据库

聊天记录存储在插件目录 `chat_history.db`（SQLite），包含字段：
- group_id: 群号
- user_id: 用户QQ
- user_name: 用户名
- message_text: 消息文本
- timestamp: 时间戳
- is_bot: 是否机器人消息

## 适配平台

- aiocqhttp（QQ 个人号 / NapCat / Lagrange 等）
